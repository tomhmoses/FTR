This is it! Our big winter mountaineering trip!

Would you like to be a part of Scotland 2008/2009?

Thanks to the amazing Phil Vincent we will still be able to run the winter skills course this year so if you have never mountaineered before then you have this amazing opportunity to learn all you need to know at a subsidised price from some ace instructors
